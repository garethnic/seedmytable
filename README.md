# SeedMyTable

This package creates a solid `DatabaseSeeder.php` file.

## Install

Via Composer

``` bash
$ composer require garethnic/seedmytable
```

## Usage

``` bash
$ vendor:publish --force
```

This will overwrite your existing `DatabaseSeeder.php` file.
